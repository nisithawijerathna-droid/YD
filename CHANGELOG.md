## [1.0.0] - 2024-XX-XX

### Added
- Initial release
- YouTube video download with multiple quality options
- Audio-only download in MP3 format
- Real-time download progress with speed and ETA
- Custom download location selection
- Video preview with thumbnail, title, and metadata
- Modern glassmorphic UI design
- Cross-platform support (Windows, macOS, Linux)
- Automatic FFmpeg handling via imageio-ffmpeg
- File size display for each quality option

### Features
- Clean and intuitive user interface
- Responsive design with smooth animations
- Support for various video resolutions (1080p, 720p, 480p, etc.)
- Error handling and user feedback
- Progress bar with percentage, speed, and ETA indicators

### Technical
- Built with Python and PyWebView
- Powered by yt-dlp for reliable downloads
- Glassmorphic design with backdrop filters
- Asynchronous download handling
